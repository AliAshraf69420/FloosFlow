# Suggested libraries:

## Charting Technologies for React Dashboards
- ### Recharts
	- React-specific charting library.
	- Can create bar charts, line charts, pie charts, area charts, etc.
	- https://recharts.org
- ### Chart.js (with react-chartjs-2)
	- Widely used, well-documented.
	- Offers more customization than Recharts.
	- Great for responsive charts.
	- https://www.chartjs.org
