# Grade Evaluation Criteria

| Phase                                                      | Weight  | Evaluation Criteria                                                                                                                                                                                                                                                                       |
| ---------------------------------------------------------- | ------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Phase 1: Group Team & Project Idea**                     | **5%**  | - Project idea relevance and feasibility (2%) <br> - Clear team formation and GitHub repo created (2%) <br> - Professional communication and timely submission (1%)                                                                                                                       |
| **Phase 2: UI/UX Design & Static Prototype**               | **25%** | - Completeness of required pages (7 pages, desktop + mobile) (5%) <br> - Quality of wireframes/mockups (clarity, consistency, alignment with requirements) (2.5%) <br> - Static prototype in HTML5 + Tailwind, accuracy to design (15%) <br> - Accessibility and responsive design (2.5%) |
| **Phase 3: Core React Components**                         | **30%** | - Implementation of reusable React components (15%) <br> - Proper use of React Router for navigation (5%) <br> - Basic state management (5%) <br> - Code quality, responsiveness, and adherence to UI design (5%)                                                                         |
| **Phase 4: Advanced Frontend Features & Data Integration** | **30%** | - Integration with public APIs or mock API (10%) <br> - Proper data fetching and rendering in UI (10%) <br> - CRUD functionality (5%) <br> - Error handling and loading states (5%)                                                                                                       |
| **Phase 5: Final Delivery & Presentation**                 | **10%** | - Fully working full-stack application (4%) <br> - Project report (3%) <br> - Presentation/demo (3%)                                                                                                                                                                                      |
